# gatsby-remark-obsidian

> Gatsby plugin to support Obsidian markdown syntax

## Installation

```bash
npm install gatsby-remark-obsidian
```

## Usage

Add the plugin to your Gatsby config:

```js
{
  resolve: `gatsby-plugin-mdx`,
  options: {
    gatsbyRemarkPlugins: [
      {
        resolve: `gatsby-remark-obsidian`,
      },
    ],
  },
}
```
